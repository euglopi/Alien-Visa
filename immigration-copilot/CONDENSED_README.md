# Condensed Immigration Copilot

All functionality merged into minimal files with no Unicode characters.

## Files
- `immigration_copilot.py` - Single file containing CLI, Web, and O-1 functionality
- `templates/condensed_index.html` - Minimal web interface
- `condensed_requirements.txt` - Dependencies
- `run_condensed.py` - Run script

## Installation
```bash
pip install -r condensed_requirements.txt
```

## Usage

### CLI Mode (Default)
```bash
python immigration_copilot.py
# or
python run_condensed.py cli
```

### Web Mode
```bash
python immigration_copilot.py web
# or
python run_condensed.py web
# Open http://localhost:3000
```

### O-1 Specific Mode
```bash
python immigration_copilot.py o1
# or
python run_condensed.py o1
# Open http://localhost:3000
```

## Features
- CLI assessment with questions
- Web chat interface
- O-1 eligibility scoring
- Immigration pathway recommendations
- No Unicode characters
- Minimal file count
